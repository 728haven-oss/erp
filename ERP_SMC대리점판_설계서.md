# SMC 대리점 전용 ERP 설계서 (v4 장비사 ERP와 별개 제품)

**작성일:** 2026-07-04 · **지시:** "V4는 장비사 위주 ERP. 지금 하려는 건 SMC 대리점만을 위한 ERP." + 송수신 요구: 견적서 송출(가격·재고 송신 / 구매·판매가 송출), 자재 입고 데이터 SMC로부터 수신.
**벤치마킹 근거:** 히트판21c SMC판 실물(SMC자동입고분관리, ▷한국SMC(주)◁ 메뉴), 실백업 SMC_* 테이블 해독(발주/수주/인보이스/납품스케줄/대체품/승인), 귀사 실데이터(품목 46,846·거래처 2,373).

## 2. SMC 송수신 명세 (요구 3종)

### 송신 ① 견적서 송출 — 가격·재고 포함
- 견적서 화면 [SMC 송출] → 송출함에 문서 생성: 품번(SMC 규격)·수량·**판매단가·현재고** 포함.
- 형식: 표준 CSV/엑셀(품번,품명,규격,수량,단가,재고,납기요청) + 메일 첨부 송부(mailto/기록). 실서비스: SMC 채널 협의(EDI/API) 후 자동 전송.

### 송신 ② 가격 데이터 송출 — 구매가·판매가
- 품목(선택 or 전체 필터) → [가격표 송출]: 품번·구매가(매입)·판매가·적용일 CSV 생성+송부 기록. 수신처(SMC 담당/지점) 주소록 연동.

### 수신 ③ 자재 입고 데이터 (SMC → 우리)
- **SMC 수신함** 메뉴: SMC가 내려주는 입고/출하 자료 업로드(CSV/엑셀 붙여넣기) → 해독된 SMC_* 매핑으로 파싱:
  발주번호(SPNO)↔우리 발주 자동 매칭, 수주번호(SUJUNO) 연계, 인보이스번호(INVNO), 납품예정 회차(NAPDT/NAPSEQ/NAPQTY), **대체품번(ALT1~4)** 처리(원품번↔대체품번 치환 확인 팝업), 승인(SMCOK).
- 매칭 성공분 → [자동입고] 일괄 처리(입고 전표 생성, 백오더 대기 명세서 자동 해제 알림, 원가 자동 삽입 연동).
- 미매칭분 → 보류 리스트(수동 매칭 UI).

### 수신 ④ 견적 작성 중 품번 → SMC 가격 데이터 수신 (지시 추가분)
- 견적서 라인에 품번 입력 시 SMC 가격을 조회해 단가 제안: 테스트판은 **SMC 가격표 수신함**(SMC가 배포하는 가격 파일 업로드 → `DB.smcPrices` 적재)에서 즉시 조회(품번 자동완성 시 "SMC가 12,500 · 재고문의" 뱃지 + [적용] 버튼). 실서비스는 SMC 실시간 조회 API/채널 협의 후 같은 자리에 연결.
- 대체품번(ALT) 등록 시 원품번 입력해도 대체품 가격 함께 표시.

### 실통신 경계 (비용/협의 명기)
파일 기반은 무료·즉시 가동. 실시간 연동은 SMC코리아 대리점 지원과 인터페이스(EDI/API/포맷) 협의 필요 — 협의 결과에 따라 Flask 단계에서 구현(자리 준비). 히트판 수신 파일 샘플 확보 시 포맷 100% 일치 구현.
