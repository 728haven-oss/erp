# 연동(카드·홈택스·은행) 비용 및 타사 설치 과금·비용처리 가이드

**작성일:** 2026-07-04 · **근거:** 팝빌 공식 요금표(2026-07 조회), CODEF/쿠콘·오픈뱅킹 공개 정보, 기존 "비용 경계표"와 연계

## 1. 연동별 실제 비용 (중계사 팝빌 기준 — 공식 요금)

| 연동 | 방식 | 비용 (VAT별도) | 비고 |
|---|---|---|---|
| **홈택스 전자세금계산서** (정발행·역발행 전송) | 팝빌 API | **200원/건** | 가입비·기본료 없음, 포인트 충전제(최소 11,000원). 우리 v4의 건당 110원 기본값은 200원으로 조정 권장 |
| **홈택스 매입자료 수집** (매입 세금계산서 자동 수집) | 팝빌 홈택스수집 | **월 30,000원** | 현금영수증 수집도 별도 월 30,000원 |
| **은행 계좌 거래내역 조회** (통장 자동 대사) | 팝빌 계좌조회 | **월 5,000원/계좌** | 현재 CSV 무료 대체 가동 중 → 전환 시 이 요금 |
| **카드 매입내역 자동수집** | CODEF/쿠콘 API | **비공개(협의)** — 통상 월 수만원대 + 건당 과금 구조 | 공식가 미공개라 계약 전 견적 필수. 당분간 카드사 CSV 업로드(무료)로 충분 |
| 예금주 조회 (오입금 방지) | 팝빌 | 성명 50원/건, 실명 80원/건 | 통장 대사 보조 |
| 알림톡(카카오) | 팝빌 | **12원/건** | 결재·납품 알림용. SMS 20원, LMS 50원 |
| 팩스 발송 | 팝빌 | 50원/장 | 구세대 거래처 대응용 |
| 오픈뱅킹(공식 API) | 금융결제원 | 핀테크사업자 등록 필요 — 소기업 부적합 | 스크래핑(팝빌/CODEF)이 현실적 |

**월 예상 실비 시나리오 (계산서 월 50건 + 계좌 1개 + 매입수집 사용 시):**
50건×200원 = 10,000 + 계좌 5,000 + 홈택스수집 30,000 = **월 45,000원 수준**.
알림톡 월 300건 추가해도 +3,600원. → 소기업 기준 월 5만원 내외가 상한선.

## 2. 우리 ERP에 반영된 구조
- 설정>홈택스 연동 카드: 중계사·API키·건당 비용 입력 → 발행 시 자동 비용 기록·월/연 집계 (v4 기구현). 계좌·카드도 같은 패턴으로 확장 예정.
- 모든 연동은 "자리+시뮬레이션" → 실서비스(Flask) 전환 시 중계사 계약만 붙이면 됨.

## 3. 타사 설치 시 과금 방법 (권장 모델)

### 요금 구성 3층
1. **초기 구축비 (1회)**: 설치 + 히트판 등 기존 데이터 이관 + 교육. 권장: 에디션별 50만~150만원 (이관 데이터량·커스터마이징에 따라 견적).
2. **월 구독료**: 에디션(Lite 3만/Standard 5만/Pro 8만, 2인 포함 + 인당 추가) — 상세는 「ERP_요금제_에디션설계.md」.
3. **연동 실비 (per-use)**: 위 §1 항목 — **실비 그대로 전가** 원칙.

### 연동 실비를 받는 2가지 방식 (비용처리 포함)

| | A안. 고객사 직가입 (권장) | B안. 우리가 통합 계약 후 재청구 |
|---|---|---|
| 구조 | 고객사 명의로 팝빌 가입, 고객사가 직접 충전 | 우리가 팝빌 파트너(연동회원) 계약, 고객사 사용량을 우리가 결제 |
| 우리 매출 | 구독료·구축비만 | 구독료 + 연동 실비(마진 가능) |
| 세무 처리 | 각자 비용 처리(팝빌이 고객사에 계산서 발행) — 우리 정산 업무 없음 | 우리가 고객사에 사용량 포함 세금계산서 발행(매출), 팝빌 비용은 우리 매입 — 월 정산·미납 리스크 관리 필요 |
| 장점 | 투명, 관리 제로, 분쟁 없음 | 단가 협상력(볼륨), 마진, 고객은 창구 일원화 |
| 권장 | **초기(고객 1~5개사)** | 고객 10개사 이상, 팝빌 파트너 단가 협의 성립 시 전환 |

### 계약서 필수 명기 사항
- 연동 실비는 중계사 요금 변동 시 연동 조정됨.
- 국세청 전송 실패·중계사 장애는 중계사 SLA 따름(우리 면책).
- 데이터 소유권은 고객사, 해지 시 전체 백업(JSON/MDB호환) 제공 — 히트판 인수인계와 동일 원칙.
- 공동인증서(사업자범용, 연 10만원 내외)는 고객사 부담.

## 4. 손익 예시 (Standard 5인 고객 1개사)
월 구독 86,000 + 연동 실비 pass-through(45,000, A안이면 0) → 우리 월 매출 86,000원/사.
구축비 100만원(이관 포함) 1회. 10개사 확보 시 월 86만원 + 유지보수.
